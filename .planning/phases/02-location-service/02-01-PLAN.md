  <files>location-service/go.mod, location-service/go.sum, location-service/internal/kafka/producer.go</files>
  <behavior>
    - Kafka producer wraps franz-go kgo.Client, exposes Publish(ctx, key, value) error
    - Publish sends a record to the "driver.location" topic synchronously (ProduceSync), keyed by driver_id
    - Publish returns error if ProduceSync fails — caller maps to HTTP 503
    - Ping uses kadm.ListBrokers to verify Kafka connectivity — returns error if broker unreachable
    - kgo.Client configured with: SeedBrokers from KAFKA_ADDR env, LeaderAck, DisableIdempotentWrite, RecordDeliveryTimeout(5s)
    - The Kafka topic name is read from KAFKA_TOPIC env (default "driver.location")
    - Producer interface is defined in the same file for test mocking
  </behavior>
  <action>
    Run from the project root (not inside location-service/):

    1. Create location-service/go.mod:
       ```
       module mototaxi/location-service

       go 1.24
       ```

    2. Install dependencies (run inside location-service/):
       ```bash
       cd location-service
       go get github.com/go-chi/chi/v5
       go get github.com/twmb/franz-go/pkg/kgo
       go get github.com/twmb/franz-go/pkg/kadm
       go get github.com/redis/go-redis/v9
       go get github.com/prometheus/client_golang/prometheus
       go get github.com/prometheus/client_golang/prometheus/promauto
       go get github.com/prometheus/client_golang/prometheus/promhttp
       go get github.com/alicebob/miniredis/v2
       ```

    3. Create location-service/internal/kafka/producer.go:
       - Define `Producer` interface: Publish, Ping, Close
       - Implement `franzProducer` struct wrapping *kgo.Client
       - `NewProducer(addr, topic string)` returns (Producer, error) — addr = broker address, topic = topic name
       - kgo.NewClient options: SeedBrokers(addr), RequiredAcks(kgo.LeaderAck()), DisableIdempotentWrite(), RecordDeliveryTimeout(5*time.Second)
       - Publish: create &kgo.Record{Topic: p.topic, Key: []byte(key), Value: value}; call cl.ProduceSync(ctx, record).FirstErr()
       - Ping: create kadm.NewClient(p.cl); call adm.ListBrokers(ctx); return error if call fails
       - Close: call p.cl.Close()

    Do NOT use confluent-kafka-go — it requires CGO and breaks the FROM scratch build.
    Do NOT use kgo.AllISRAcks with LeaderAck — pair LeaderAck with DisableIdempotentWrite or franz-go returns error at construction time (see RESEARCH.md Pitfall 1).
  </action>
